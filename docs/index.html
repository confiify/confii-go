Why Confii?

Type-Safe Generics

Config[T] with cfg.Typed() returning *T. IDE autocomplete, struct tag validation, and compile-time safety.

Every Source

YAML, JSON, TOML, INI, .env, env vars, HTTP, AWS S3, SSM, Azure Blob, GCS, IBM COS, Git repos.

Secret Resolution

${secret:key} placeholders resolved from AWS Secrets Manager, Azure Key Vault, GCP Secret Manager, or HashiCorp Vault.

Smart Merging

6 merge strategies (replace, merge, append, prepend, intersection, union) with per-path overrides.

Full Introspection

Explain() shows where values came from. Source tracking, override history, conflict detection, debug reports.

Config Composition

Hydra-style _include and _defaults directives with cycle detection. Environment-aware default + prod merging.

Drift Detection

Diff configs, detect unintended changes, snapshot versions, and rollback. Built-in observability and metrics.

Dynamic Reload

File watching via fsnotify with incremental reload. Change callbacks. Freeze for immutability.

Production Ready

Thread-safe (sync.RWMutex), zero global state, 96%+ test coverage, CI/CD, govulncheck clean.

Quick Start

// Basic usage cfg, err := confii.New[any](ctx, confii.WithLoaders( loader.NewYAML("config.yaml"), loader.NewEnvironment("APP"), ), confii.WithEnv("production"), ) host, _ := cfg.Get("database.host") port := cfg.GetIntOr("database.port", 5432) // Type-safe access cfg, err := confii.New[AppConfig](ctx, confii.WithLoaders(loader.NewYAML("config.yaml")), confii.WithValidateOnLoad(true), ) model, _ := cfg.Typed() fmt.Println(model.Database.Host) // IDE autocomplete works
